# The IPWKBZ model

- *i*: interpolation
- *p*: percentage choice grids
- *w*: choose aggregate savings w = k' + b'
- *k*: choose k' given w and z
- *b*: allow for borrowing to finance k'
- *z*: some possibly persistent shock process
